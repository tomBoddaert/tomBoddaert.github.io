Site v2

A few weeks ago, I finished rewriting site, the templating program I use to build this website. This is currently the only project I maintain in go.

The two main goals for this rewrite were to make it more configurable for the end user and to make the code much neater. Although I'm not so sure about the second goal, the new version is definitly more configurable.

Site v2's main addition is the siteConfig.json file, which allows the user to set several parameters including the input and output directories, which were previously hard-coded. I also added sprig's functions, which provides many utilities that can be used directly in the templates.

This new version also comes with an updated example site to display the main features.
